# Stacklang

Stacklang is a programming language designed for working with stacks. It aims to simplify the handling of stack-based operations, making it easier to implement algorithms and data structures that rely on stacks.

## Features
- **Simple Syntax**: The syntax is straightforward, making it easy for beginners to learn.
- **Stack Operations**: Provides built-in functions for common stack operations like push, pop, and peek.
- **Performance**: Optimized for speed and efficiency in stack manipulation.

## Installation
To install Stacklang, clone the repository and build from source:

```bash
git clone https://github.com/Cosmic-crypto/stacklang.git
cd stacklang
# build instructions here
```

## Usage
To run a Stacklang program, use the following command:

```bash
stacklang your_program.stack
```

## Example
Here's a simple example of using Stacklang to reverse a string:

```stacklang
# Reversing a string
input: "hello"
output: "olleh"
```
